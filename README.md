# Landsat LST Processing

## Setup
1. Clone repository

2. Install dependencies:
<ul>
<li>*earthengine-api*</li>
<li>*geemap*</li>
<li>*numpy*</li>
<li>*matplotlib*</li>
<li>*pytest*</li>
</ul>
By running command:
`pip install -r requirements.txt`

3. Set up Google Earth Engine authentication

4. Run the main script:
```bash
python main.py
```
